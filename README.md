# 超市管理系统（进销存系统） java550

## 项目概述

本项目是基于 SpringBoot + Vue 的超市管理系统（进销存系统），采用前后端分离的架构设计。

## 技术栈

### 后端
- Spring Boot
- MySQL
- Maven
- druid
- Mybatis

### 前端
- Vue

## 功能模块

- 部门管理
- 商品管理
- 销售记录统计
- 入库出库管理

## 系统角色

- 系统管理员
- 部门经理
- 库管员
- 销售人员

## 运行环境

- 后端：Java 1.8+
- 前端：现代浏览器
- 数据库：MySQL 5.7+

## 部署步骤

1. 环境准备：确保已安装 Java 和 MySQL
2. 数据库配置：导入 MySQL 数据库文件
3. 后端部署：使用 Maven 编译项目，运行 Spring Boot 应用
4. 前端部署：在浏览器中打开前端项目
